# To Run
Run naming server
```bash
orbd -ORBInitialPort 1050 -ORBInitialHost localhost
```

run server (game) and host a game
```bash
java -cp "target/classes:target/dependency/*" tn.edu.eniso.kombla.KomblaGame
```

run client (game) and join a game
```bash
java -cp "target/classes:target/dependency/*" tn.edu.eniso.kombla.KomblaGame
```
